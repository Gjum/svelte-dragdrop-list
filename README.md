# 🔃 SortableList

This component renders a list of items which can be reordered by draggin and dropping and implements FLIP animation for adding/removing/reordering items in the list.

## ⤵️ Props

| name       | type      | required | default |
| ---------- | --------- | -------- | ------- |
| `list`     | array     | ✔️       | ✖️      |
| `listItèm` | component | ❌       |
